# DTI WEL System

World-class Workplace Experience Learning management platform for Design & Technology Institute.

## Features

- 6 role-based portals (Admin, Learner, Supervisor, Leadership, Employer, Alumni)
- 14 integrated modules
- Real-time dashboards with KPI tracking
- Aligned with ILO, UNESCO-UNEVOC, and COTVET standards
- Progressive Web App (installable, works offline)
